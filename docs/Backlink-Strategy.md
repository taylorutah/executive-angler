## Phase 4: Data-Driven Linkable Assets (Month 2-3)

Create content so valuable that other sites link to it naturally.

### Asset Ideas
1. **Annual "State of Fly Fishing" Report** — Aggregate catch data from journal users: top rivers, peak months, most productive flies. Fishing publications will cite this.
2. **Embeddable Hatch Chart Widget** — Free embeddable hatch chart for any river. Other fishing blogs embed it, linking back to you.
3. **Interactive River Map** — A Mapbox-powered map showing all 85+ rivers with conditions. Embed code available for fishing forums.
4. **Species Size Distribution Data** — Chart showing catch sizes by species from journal data. Unique dataset no one else has.
5. **"Best Time to Fish" Calculator** — Enter a river, get optimal months based on historical data.

### Infographic/Visual Assets

## Do's and Don'ts

### DO
- Participate genuinely in fishing communities before dropping links
- Create genuinely useful content that earns links naturally
- Personalize every outreach email — mention specific content on their site
- Follow up once after 7 days, then move on
- Track every outreach attempt in a spreadsheet
- Diversify anchor text (brand name, naked URL, generic, keyword)

### DON'T
- Buy links or participate in link schemes
- Spam forums with self-promotional links
- Use automated link building tools
- Over-optimize anchor text (Google penalizes this)
- Ignore nofollow links — they still signal brand presence
- Give up after rejection — expect 80%+ rejection rate on guest posts
